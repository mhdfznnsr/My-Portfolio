My Projects

Hand Sign Detection

Hand Sign Detection is a project based on classification technique. It uses a pre-trained model and uses it's neural network architecture to train a custom build dataset. Through Transfer Learning, a custom build dataset consisting of 160 images and their XML files are used for training purpose.

Movie Dataset Creation

It is a Data Science project in which information about the movies from Wikipedia is Web Scraped and Cleaned for analysis purposes. It uses different libraries such as BeautifulSoup, Pandas, requests, pickle etc for various extraction and cleaning.

Sales Analysis

Information about the sales of a business is vast. Through Data Science, the data is analysed and the following questions were answered: What was the best month for sales? How much was earned that month? What products are most often sold together?

Text Extraction And Analysis

Data Analytics is a field of data extraction and analyzing the data. This DA project focuses on web scraping and analyzing the data to fill the required attributes. It uses BeautifulSoup for web scraping and NLP for Sentiment Analysis.

Reviews Sentiment Analysis

This project is based on Text Vectorization and Bagging. From different Classification Algorithms such as SVM, Logistic Regression, Gaussian Naive Bayes and Decision Tree, the algorithm with best performance is taken and used for prediction of reviews.
